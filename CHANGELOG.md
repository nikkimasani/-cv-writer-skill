## Version History

### Versioning Scheme

We use [Semantic Versioning](https://semver.org/):
- **MAJOR** version for incompatible changes
- **MINOR** version for added functionality (backwards compatible)
- **PATCH** version for bug fixes (backwards compatible)

### How to Upgrade

Download the latest `.skill` file from [Releases](https://github.com/YOUR_USERNAME/cv-writer-skill/releases) and re-install in Claude to replace the previous version.
